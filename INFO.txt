# Second run
This is how to start the 0.5 release:
./run_debug.sh

Then the application will listen on localhost:5000

If it is the first time you start the application, you must create the sqlite database:
export FLASK_APP=main.py
flask initdb

# First run
This is how to start the hello world demo:
$ export FLASK_APP=mainp.y
$ export FLASK_DEBUG=1
$ flask run
